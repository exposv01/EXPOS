# EXPOS

EXPOS is an internal operational system for EXP Gaming. It is planned as a lightweight operations platform built around Google Workspace.

This repository currently contains the initial project architecture only. Business features are intentionally not implemented yet.

## Planned Foundation

- Google Sheets is the One Source of Truth for operational data.
- Google Apps Script is the business logic layer.
- Google Drive is the file storage layer.
- Gmail and WhatsApp are notification channels for future tasks.

## Current Scope

Version `0.1.0` establishes the repository structure, documentation starting points, and Apps Script file boundaries.

The following modules are reserved for future implementation:

- Absensi
- Izin
- Kasbon
- Report Problem

The following areas are not part of this foundation phase:

- Login
- Dashboard
- Payroll
- Inventory
- Approval
- Analytics
