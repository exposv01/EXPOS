# UI-007 — Home Dashboard Mobile

Tanggal delivery: 2026-07-04  
Sprint: Sprint 4 Part 2  
Status: Implemented in GitHub  
Repository: exposv01/EXPOS

## Objective

Membuat halaman Home Dashboard mobile-first untuk EXPOS v0.1 dengan standar lebar 390px iPhone 16.

UI-007 berfungsi sebagai pintu masuk Modul 1, bukan dashboard kompleks.

## Bible Compliance

Acuan implementasi:

- Google Sheets tetap menjadi database operasional.
- Google Apps Script tetap menjadi business logic dan automation layer.
- GitHub hanya menjadi source control untuk source code.
- UI hanya presentation layer.
- Workflow tetap melalui service/controller.
- Tidak membuat database baru.
- Tidak menambah scope dashboard kompleks.
- Tidak mengaktifkan Bell Notification sebagai fitur operasional v0.1.

## Scope UI-007

### Included

1. Mobile shell dengan max-width 390px.
2. Header EXPOS.
3. Greeting card dengan tanggal dan jam WIB.
4. Banner budaya kerja HAPPY:
   - Happy Inside
   - Always Improve
   - Positive Attitude
   - Play as One Team
   - Yes We Can
5. Menu utama Modul 1:
   - Report Problem
   - Absensi
   - Kasbon
   - Pengajuan Izin
   - Rekap Absensi
6. Ringkasan Absensi Hari Ini:
   - Hadir
   - Izin
   - Kasbon
   - Problem Open
   - CBN / ARH / SLW
7. Hook aman ke Apps Script controller:
   - `openExposModule(moduleName)`
   - `getHomeDashboardSummary()`
8. Fallback aman jika controller belum tersedia.

### Excluded

1. Login.
2. Approval.
3. Payroll.
4. Inventory.
5. Dashboard kompleks.
6. Bell Notification aktif.
7. Analytics.
8. Telegram / Discord.
